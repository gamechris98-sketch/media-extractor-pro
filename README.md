# Media Extractor Pro Web (Static Edition) 🚀

Premium web-based media downloader optimized for **GitHub Pages**. Featuring a sleek iOS-inspired Glassmorphism UI and powered by high-performance Open APIs.

[한국어 설명은 아래에 있습니다]

## Key Features
- **Modern UI**: iOS 17 style Glassmorphism with smooth animations.
- **Full Responsive**: Perfectly optimized for PC, Tablet, and Mobile browsers.
- **No Backend Required**: Runs directly in your browser via GitHub Pages.
- **Open API Powered**: Uses high-performance external APIs for fast media extraction.
- **Batch Processing**: Download multiple videos/audios sequentially.

## How to Use
Simply visit the [GitHub Pages URL](https://gamechris98-sketch.github.com/media-extractor-pro) (Once activated).

---

## 주요 기능
- **모던 UI**: iOS 17 스타일의 Glassmorphism과 부드러운 애니메이션 적용.
- **반응형 디자인**: PC, 태블릿, 모바일 모든 환경에 최적화된 화면.
- **백엔드 불필요**: GitHub Pages에서 별도의 서버 설치 없이 즉시 실행 가능.
- **오픈 API 활용**: 고성능 외부 API를 연동하여 빠르고 안정적인 추출 지원.
- **일괄 처리**: 여러 URL을 한 번에 입력하여 연속 다운로드 가능.

## 사용 방법
활성화된 [GitHub Pages 주소](https://gamechris98-sketch.github.com/media-extractor-pro)에 접속하기만 하면 됩니다.
